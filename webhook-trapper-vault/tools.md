# Webhook Trapper

A full-stack webhook interception and management platform. Capture incoming webhooks, filter and transform payloads, forward to destinations with retry logic, and monitor activity in real-time.

## Tools

**Languages:** TypeScript, React
**Runtime / Platform:** Node.js 20+, browser (React 18)
**Key third-party dependencies:** Express, React Router, MUI, Drizzle ORM, better-sqlite3, Redis (ioredis), BullMQ, JWT, bcryptjs

## Purpose

Webhook Trapper allows developers and integrations to intercept webhook requests from external services, apply filtering and transformation rules, forward to multiple destinations with retry policies, and inspect traffic. Useful for testing, debugging, monitoring, and routing webhooks in complex integration scenarios.
